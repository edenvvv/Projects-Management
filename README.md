# Projects-Management

## <a href="https://projects-management-ade.herokuapp.com/">Live Demo</a>

## Requirements:

- Node.js 14.16.0 and up <br/>
- MongoDB 4.4 and up

## Modules requirements:

`npm install express`<br/>
`npm install nodemon`<br/>
`npm install chai-http`<br/>
`npm install ejs`<br/>
`npm install mongodb`

## Run:

`npm run dev`
